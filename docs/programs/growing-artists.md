# Growing Artists Program

The growing artists program is available to all users on our platform. It is designed to help users wishing to join the verified artists program be able to reach the requirements necessary to be eligible.

### Requirements

- None!

### Features

- Detailed tips on how to grow your reach
- Access to help from existing verified artists [^7]
- ... and more!

[^7]: This does not include all verified artists, only those who have partnered with us internally.
